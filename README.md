# EDA sobre o dataset Covid-19 Global Dataset
## introdução:
Covid-19 foi uma doença causada pelo corona vírus SARS-CoV-2. descoberta em dezembro de 2019 na cidade de Wuhan, china. devido a maneira como essa doença e transmitida, ela se espalhou rapidamente pelo mundo, e por isso, em março de 2020, a Organização Mundial da Saúde(OMS) declarou a pandemia global.</br>

durante poucos tempo de pandemia, a COVID gerou milhões de casos de morte por todo o planeta, e isso, gerou grandes impactos Sociais, econômicos e de saúde publica. por conta disso, irei realizar uma analise sobre o dataset "Covid-19 Global Dataset" para ter uma melhor compreensão sobre a Covid-19.</br>
